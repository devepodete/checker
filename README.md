# checker
Java-checker for olympiad programming (so far linux only)

Main запускает в отдельном процессе ProgramTesting.Checker, считывает команды из терминала и отправляет их при помощи сокетов на чекер.
Чекер, получив команду, собирает исходный код и проверяет работу на тестах. Большинство действий чекера логируется. После окончания тестирования в папке build текущей посылки находятся вердикт, время и результат выполнения на тестах, логи компиляции и ошибки выполнения, если такие имеются.

Формат команд: `COMMAND [ARGS]`.
Список доступных команд на текущий момент:
  - `check contest problem submit` - проверить посылку
  - `exit` - выход
